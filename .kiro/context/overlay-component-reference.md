# Overlay Component Reference

Use this file as a text substitute for OVERLAY.png when Kiro cannot read the image.

## Buttons
- Secondary: teal background, white text.
- Primary/active: dark navy background, white text.
- Outline: white background, navy text, border.
- Inactive/disabled: grey background, muted text, disabled cursor/no action.
- Icon button: dark navy background, centered icon + label.
- Critical/destructive: pale red background, red border/text.
- Secondary action: pale cyan background, cyan text/border.

## Inputs
- Default: white background, light border, label above.
- Focus/typing: teal border and subtle focus ring.
- Filled: white background and normal border.
- Error: red border with red helper message below.
- Disabled: muted grey background and muted text.

## Menus and Dropdowns
- White rounded card, subtle border/shadow.
- Item height around 43px.
- Selected item has light grey/blue fill.
- Destructive menu actions use red text and/or pale red background.

## Status Badges
Use shared variants for: Not Started, In Progress, Completed, Overdue, Inactive, Deactivated, Ending Soon, On Track, Expired.

## Success Modal
Use centered modal/card, green success icon, bold title, helper text, and navy primary action.
